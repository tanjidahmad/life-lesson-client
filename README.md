# Overview

Digital Life Lessons is a modern full-stack platform where users can create, discover, and preserve meaningful life experiences. The platform enables community-driven knowledge sharing while providing secure authentication, premium content access, personalized dashboards, and administrative content moderation.

This repository contains the frontend application built with **Next.js**, **HeroUI**, **Tailwind CSS**, and **Better Auth**.

---

# Key Features

* Secure authentication with Better Auth
* Email & Password Authentication
* Google Authentication

* Public & Premium Lesson System
* Stripe Premium Upgrade

# Role-Based Access

## Guest User

* Browse all public life lessons
* Search lessons by title
* Filter lessons by category and emotional tone
* View featured lessons
* Register and Login
* View premium lesson preview (blurred content)

---

## Registered User

Everything available to Guest Users, plus:

* Create new life lessons
* Edit own lessons
* Delete own lessons
* Change lesson visibility (Public / Private)
* Save & remove favorite lessons
* Like / Unlike lessons
* Comment on lessons
* Report inappropriate lessons
* Access personal dashboard
* Manage personal profile
* View lesson statistics
* Upgrade to Premium

---

## Premium User

Everything available to Registered Users, plus:

* Create Premium lessons
* Access Premium lessons created by other Premium members
* Premium badge
* Lifetime Premium access after successful Stripe payment

---

## Administrator

Complete platform management capabilities:

* View platform statistics
* Manage all users
* Promote users to Admin
* Manage all lessons
* Delete inappropriate lessons
* Mark lessons as Featured
* Mark lessons as Reviewed
* Review reported lessons
* Ignore or remove reported content
* Access full administrative dashboard
